# Requirements

## User Request

- Dashboard should show only:
  - Total available
  - Spent
  - Income
  - Cash in hand
  - Bank account
  - A small graph
- Each sidebar item should show only its own section content.
- Clicking one sidebar item should not show extra content from other sections.

## Constraints

- Keep the existing backend data model.
- Keep the app dependency-free.
- External Gemini/Claude CCG wrapper is unavailable in this environment.
